# To-Do List Web Application

A simple and elegant **To-Do List App** built using **React** and **Bootstrap**. Easily manage your daily tasks with features like add, delete, and mark as complete.

## ✨ Features

- Add new activity
- Mark activity as completed
- Delete activity
- Responsive UI with Bootstrap
- Dynamic activity counter
- Clean and modern design

## 🛠 Tech Stack

- [React](https://reactjs.org/)
- [Bootstrap](https://getbootstrap.com/)
- CSS (Custom styling alongside Bootstrap)
